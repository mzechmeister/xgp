# xgp - tcl/tk frontend gui for gnuplot

 <img src="xgp_demo.png" width="100%"/>	

xgp provides a GUI for gnuplot. It is a standalone tcl/tk script.

It is a feasibily demo for some nice-to-have features. It is largely functional but under development.
It aims for faster scripting.

### Installation

```bash
wget https://raw.githubusercontent.com/mzechmeister/xgp/master/xgp
./xgp
```

### Features
* gnuplot help via buttons
* tk embed
    * pan with mouse-wheel pressed
    * mouse zooming via rectangle, horizontal or vertical drag on x or y-axis, resp.
    * edit axis label by clicking into the axis
